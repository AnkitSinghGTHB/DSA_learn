# 🛋️ Couch to Coder: 2-Month DSA Roadmap
**A zero-stress, everyday guide to mastering Data Structures and Algorithms in Java.**

Welcome to the Couch to Coder repository! This curriculum is designed for absolute beginners who want to reach job-market readiness without getting bogged down by heavy math or intimidating jargon.

## 📁 Repository Structure

### 📚 The Concepts
These modules contain the core theory, real-world analogies, and foundational Java templates for each week.
- `week1_arrays_big_o.md`
- `week2_hash_maps_sets.md`
- `week3_linked_lists_two_pointers.md`
- `week4_stacks_queues.md`
- `week5_trees_recursion.md`
- `week6_graphs.md`
- `week7_sliding_window_greedy.md`
- `week8_dynamic_programming.md`

### 💡 The Solutions
Inside the `solutions/` directory, you will find the fully coded Java solutions for every single LeetCode practice problem assigned in the roadmap. These solutions are meant to be a reference.

## 🧠 How to Use This Repo
1. **Read the Concept Module**: Spend 10-15 minutes reading the weekly concept file to understand the analogy and the Java syntax.
2. **Attempt the Problems**: Open LeetCode and try the assigned problems. 
3. **The 45-Minute Rule**: Do not spend more than 45 minutes completely stuck on a problem! If you are stuck, look at the solutions provided in the `solutions/` folder, understand the logic, and then type out the code yourself to build muscle memory.

Good luck on your journey to becoming a coder! ☕💻
