# 🛋️ Couch to Coder: 3-Month DSA Roadmap
**A zero-stress, everyday guide to mastering Data Structures and Algorithms in Java.**

Welcome to the Couch to Coder repository! This curriculum is designed for absolute beginners who want to reach job-market readiness without getting bogged down by heavy math or intimidating jargon. We've expanded this to cover the full **LeetCode Top Interview 150** list, ensuring you're ready for any big-tech interview.

### 💡 The Solutions
Inside the `solutions/` directory, you will find the fully coded Java solutions for every single LeetCode practice problem assigned in the roadmap. 
- `solutions/week1_solutions.md` to `solutions/week12_solutions.md`

## 🧠 How to Use This Repo
1. **Read the Concept Module**: Spend 10-15 minutes reading the weekly concept file to understand the analogy and the Java syntax.
2. **Attempt the Problems**: Open LeetCode and try the assigned problems. 
3. **The 45-Minute Rule**: Do not spend more than 45 minutes completely stuck on a problem! If you are stuck, look at the solutions provided in the `solutions/` folder, understand the logic, and then type out the code yourself to build muscle memory.

Good luck on your journey to becoming a coder! ☕💻
